# User-authetication-authorization-system
This system uses some of the basic user authentication &amp; authorization guidlines. I have built this system using technologies like Node js, Express js, EJS, bcrypt and MySql.  Using this system user can register with email id and password.  This syetem encrypts your password while storing it in DB, hence either hacker or developer himself cannot see your password in database.  I have made use of JWT to do user authorization and even if someone tries to temper the cookies they will be logged out instantly.
